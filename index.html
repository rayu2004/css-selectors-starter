CSS Selectors Introduction

This week, we’re beginning to learn CSS! We use CSS to style our pages beyond the defaults added by the browser. Understanding how the different selectors work and being able to apply them with the correct syntax is important for developers and for your homework and final project in the course. Be sure to follow along as we work, and always have your HTML file open in the browser while you style it and keep the dev tools open as well, as this will help us catch issues and test out other settings in our styles.
